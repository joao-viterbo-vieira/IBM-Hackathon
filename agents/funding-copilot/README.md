# Funding Copilot Agent

This agent helps SMEs find, filter, and draft funding applications.

## Setup

1.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

2.  Run the MCP server:
    ```bash
    python tools.py
    ```

## Usage

The agent uses the `funding-copilot-tools` MCP server to access funding data and save drafts.
